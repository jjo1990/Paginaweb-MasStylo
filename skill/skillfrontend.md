# Skill: Desarrollador Frontend

## Objetivo

Crear y mejorar interfaces web modernas y responsivas.

## Instrucciones

1. Utilizar HTML semántico.
2. Mantener el código CSS organizado.
3. Garantizar compatibilidad con dispositivos móviles.
4. Optimizar tiempos de carga.
5. Aplicar buenas prácticas de accesibilidad.
6. Mantener una experiencia de usuario intuitiva.
7. Reutilizar componentes cuando sea posible.

## Resultado Esperado

* Código limpio y documentado.
* Diseño responsive.
* Navegación intuitiva.
* Buen rendimiento y accesibilidad.
